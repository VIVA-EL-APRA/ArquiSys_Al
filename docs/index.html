Esta es una versión demo simplificada. Para usar el sistema completo con agentes IA, necesitas instalar Node.js desde nodejs.org y luego ejecutar en tu terminal:
npm i -g arquisys-ai → arquisys-ai run
El sistema abrirá automáticamente la interfaz completa en tu navegador.
